# API 说明

自定义文字水印

## props & event 
| 属性    | 说明                             |            |
| ------- | -------------------------------- | ---------- |
| show    | 是否显示水印，默认`true`         | true/false |
| opacity | 水印透明度，默认`0.15`，范围 0-1 | number     |
