 # Vue Axios Composition Util 

 `npm i axios` 

 1. 支持防抖
 2. 拓展请求可取消
 3. 错误自动捕获，数据自动响应
 4. 拓展流文件下载插件
 5. 级联请求（未开发、计划）
 
##  创建一个axios 实例
 
`createAxios`

##  返回参数 

`server`
`useAxiosRequest`
